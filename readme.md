### 自动添加一行在匹配的行数

```
sed -i  "/exit/i\allow chengyongxu.com" hello.conf
```

> i 插入内容 sed ‘/匹配词/i\要加入的内容’ example.file 将内容插入到匹配的行目标的上一行位置）
> 在 exit 中插入 allow chengyongxu.com ，在 hello.conf 文件中
